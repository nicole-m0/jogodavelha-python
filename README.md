# 🎮 Jogo da Velha em Python
Meu Hackathon 01
Desafio 02

## 📌 Descrição

Jogo da velha desenvolvido em Python para execução no terminal, com foco na prática de lógica de programação, estruturas condicionais e repetição.

---

## ⚙️ Como executar

### Requisitos

* Python 3
* Biblioteca `colorama`

### Instalação

```bash
pip install colorama
```

### Execução

```bash
python jogodavelha.py
```

---
### 💡 Observações
* Funciona em Windows, Linux e Mac
* Executado via terminal (sem interface gráfica)

## 🚀 Funcionalidades

* Validação de vitória (linhas, colunas e diagonais)
* Detecção de empate (Deu velha)
* Reinício da partida
* Validação de entrada (0 a 2)
* Impedir jogadas em posições ocupadas
* Uso de cores no terminal
